🚀 Next Steps to Launch

Step 1: Generate Application Key

This is required for Laravel to run properly.

php artisan key:generate

Step 2: Generate JWT Secret

This is required for JWT authentication to work.

php artisan jwt:secret --force

Step 3: Install Frontend Dependencies

Install Node.js packages and build assets.

npm install
npm run dev

Step 4: Start Development Server

Launch the Laravel development server.

php artisan serve
💡 Tip: After running the above commands, your application will be available at http://localhost:8000
